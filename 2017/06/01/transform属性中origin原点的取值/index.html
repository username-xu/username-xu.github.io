transform属性中origin原点的取值:

取值1 :

原点在元素的正中心!

取值2 :

原点在元素顶部的中间！

取值3 :

原点在元素右边的中间！

取值4 :

原点在元素底部的中间！

取值5 :

原点在元素左边的中间！

取值6 :

原点在元素左上角！

取值7 :

原点在元素右上角！

取值8 :

原点在元素右下角！

取值9 :

原点在元素左下角！
